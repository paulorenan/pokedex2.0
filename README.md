
# Pokedex 2.0

## WebSite
[Pokedex2.0](https://pokedex2-0.vercel.app/)

## Stack utilizada

**Front-end:** ReactJs, Context API

**Back-end:** NodeJs

## Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/paulorenan/pokedex2.0.git
```

Entre no diretório do projeto

```bash
  cd pokedex2.0
```

Instale as dependências

```bash
  npm install
```

Inicie o servidor

```bash
  npm start
```
